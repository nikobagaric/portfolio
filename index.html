    BACK-END

    I primarily focus on back-end development. I have experience with Node.js, Django and Spring Boot. I am also familiar with databases such as PostgreSQL and MySQL, and with tools like Docker and Git.

    Restaurant ordering API

    A simple API for ordering food from a restaurant. The API is made in Spring Boot, and was made as part of my 2024 internship. It uses a PostgreSQL database to store the data, along with Docker to ensure that the application runs on any machine.

    Video summarizer API

    Using LangChain and Whisper, I created an API that can summarize videos. The API uses AI to generate a summary of the video and returns it in a JSON format. It saves the summary in a database in the form of notes. The backend is based on Django.

    Recipe website API

    A decently robust API for a recipe website. The API is made in Django and uses a PostgreSQL database along with Docker. The API allows users to create, read, update and delete recipes, as well as to upload images for the recipes. It also has user authentication.
